[55. 跳跃游戏 - 力扣（LeetCode）](https://leetcode.cn/problems/jump-game/description/)

#### 思路

计算`maxLen = i + nums[i]`，如果`maxLen > size`就返回true，不然就结束遍历返回false;

#### 题解

```c++
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int maxLen = nums[0];
        int n = nums.size();
        for(int i = 0; i <= maxLen; ++i) {
            if(i + nums[i] > maxLen) {
                maxLen = i + nums[i];
            }
            if(maxLen >= n - 1) {
                return true;
            }
        }
        return false;
    }
};
```
